# 오늘의 골프 — 연재 기획

## 블로그 개요

- 이름: 오늘의 골프
- 발행 주기: 매일
- 형식: 그날 이슈된 골프 뉴스 3~5개를 간략하게 설명

## 뉴스 카테고리

| 카테고리 | 설명 |
|----------|------|
| 투어 | PGA·LPGA·KPGA·KLPGA 성적, 대회 일정, 순위 변동 |
| 선수 | 부상, 이적, 발언, 인터뷰 |
| 장비·용품 | 신제품, 클럽 트렌드, 용품 이슈 |
| 업계 | 국내외 골프장, 업계 동향, 정책 |

## 포스트 구조

```
제목: [날짜] 오늘의 골프 뉴스
---
뉴스 1: [제목]
뉴스 2: [제목]
뉴스 3: [제목]
(뉴스 4~5: 해당일 이슈 있을 때)
```

## 각 뉴스 항목 작성 기준

- 분량: 항목당 150~300자
- 전체 포스트: 800~1500자
- 출처 명시: 각 뉴스 끝에 출처 표기
- 해설: 사실 전달 + 한 줄 맥락 설명

## 파일 네이밍 규칙

```
drafts/YYYYMMDD-draft.md
posts/YYYYMMDD.md
reviews/review-YYYYMMDD.md
```

## 발행 워크플로우

1. `/write [날짜] [뉴스 소재]` 실행
2. 에이전트 파이프라인 자동 처리
3. proofreader 통과 → `posts/YYYYMMDD.md` 저장
4. 발행 전 최종 확인 필요시 `/review YYYYMMDD` 실행
