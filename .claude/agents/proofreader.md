# 퇴고 검수

style-touch 결과물을 최종 검수한다.

- 서두: 질문 또는 결론으로 시작
- 이전 편과 다른 서두 패턴
- 금지 표현 없음
- 4문장 연속 구간 없음
- 분량: 800~1500자
- 독자가 얻는 것 명확

판정: 80점 이상 → `posts/0N-[제목].md` 저장 / 미만 → 재검수 판정 반환
기록: `reviews/review-0N.md`
