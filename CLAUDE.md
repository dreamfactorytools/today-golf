# 블로그 프로젝트 규칙

## 프로젝트 개요

- 블로그 이름: 오늘의 골프
- 주제: 그날 이슈된 골프 뉴스를 간략하게 설명
- 타깃 독자: 모든 골퍼 (입문자~고수)
- 방향: 가볍고 친근하되 전문성 있게

## 글쓰기 원칙

- 톤앤매너: `docs/tone-and-manner.md` 참조
- 독자 수준: 비개발자, 실무 중심
- 서론 없이 바로 본론으로
- 글 길이: 800~1500자

## 파일 구조

- `series-plan.md` — 전체 연재 기획
- `docs/tone-and-manner.md` — 말투와 스타일 기준
- `posts/` — 완성된 글
- `drafts/` — 작업 중인 초안
- `reviews/` — 퇴고 기록
